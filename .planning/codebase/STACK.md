# Technology Stack

## Frontend
*   **Framework:** Next.js (Version 16.1.6)
*   **UI Library:** React (Version 19.2.3), React DOM
*   **Language:** TypeScript
*   **Styling:** CSS Modules, Vanilla CSS (based on global and variables .css files)
*   **Linter:** ESLint

## Backend
*   **Framework:** FastAPI
*   **Server:** Uvicorn (ASGI server)
*   **Language:** Python 3
*   **Authentication:** FastAPI Users (with JWT strategy)
*   **Data Validation:** Pydantic, Pydantic Settings
*   **Testing:** Pytest
*   **HTTP Client:** HTTPX, Requests
*   **Other Libraries:** python-multipart, fastapi-mail, python-dotenv, email-validator
